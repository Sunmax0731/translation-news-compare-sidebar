# UI/UX Polish

## Closed Alpha で反映済み

- pass / warn / fail を同じ語彙で扱う。
- 警告時に次アクションを明示する。
- 手動テスト手順に入力、操作、期待結果、確認ファイルを具体化する。
- Chrome Manifest V3 拡張、popup、options、local storage を備えた検証可能な拡張パッケージ の入口を repo 内に置き、ユーザーが次の確認先を探さなくてよい状態にする。

## MVP 後の改善

- 実データ取り込み UI。
- 結果差分の比較表示。
- 手動承認コメントのテンプレート化。
- 公開先 Chrome Web Store 向けのサンプル画像と FAQ。
